Foundations

Colors

Perceptually uniform OKLCH palette with light/dark variants derived from a single hue.

Why OKLCH

OKLCH (Lightness, Chroma, Hue) is a perceptually uniform color space — equal numeric changes correspond to equal visual changes. That means tweaks like "make this 10% lighter" actually look 10% lighter. The palette generates consistent soft, strong, and hover variants from a base hue.

Brand colors

--b-primary
--b-success
--b-warning
--b-danger
--b-info

Each color has three variants:

  • --b-{name} — base
  • --b-{name}-soft — light tint (for badges, alerts, hover backgrounds)
  • --b-{name}-ink — text color when used as background

Plus --b-primary-strong for emphasized text on neutral backgrounds.

Surfaces

| Token | Use | | --- | --- | | --b-bg | Page background | | --b-surface | Card/panel base | | --b-surface-2 | Slightly elevated panel (input groups, table headers) | | --b-elevated | Cards/popovers above the page |

Text

| Token | Use | | --- | --- | | --b-text | Body text | | --b-muted | Secondary text, captions | | --b-subtle | Placeholders, disabled hints |

Borders

| Token | Use | | --- | --- | | --b-border | Default 1px lines | | --b-border-strong | Hover/emphasis lines |
