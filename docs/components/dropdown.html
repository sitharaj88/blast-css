Behavior

Without JS, <details> opens on click. The bundled JS adds:

  • Escape closes the menu and returns focus to the trigger
  • Outside click closes the menu

API

| Class | Effect | | --- | --- | | .b-dropdown | Wrapper (use a <details> element) | | .b-dropdown-menu | The popup panel | | .b-dropdown-label | Section label inside the menu |
