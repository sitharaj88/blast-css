Pagination

Page-link list with current-page indication and keyboard support.

  • ‹
  • 1
  • 2
  • 3
  • …
  • 12
  • ›
html
<ul class="b-pagination">
  <li><a class="b-page" href="#" aria-label="Previous">‹</a></li>
  <li><a class="b-page" href="#">1</a></li>
  <li><a class="b-page" href="#" aria-current="page">2</a></li>
  <li><a class="b-page" href="#">3</a></li>
  <li><span class="b-page">…</span></li>
  <li><a class="b-page" href="#">12</a></li>
  <li><a class="b-page" href="#" aria-label="Next">›</a></li>
</ul>

API

| Class | Effect | | --- | --- | | .b-pagination | List wrapper | | .b-page | Each page link | | aria-current="page" | Highlights current page |
